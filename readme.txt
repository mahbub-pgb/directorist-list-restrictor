Restricts access to certain Directorist listings and adds options for Pre Sale, For Sale, and Sold listings.

== Description ==

Directorist List Restrictor allows you to restrict access to specific listing types in Directorist. You can:

* Restrict listings for non-logged-in users.
* Show a custom login notice for restricted listings.
* Assign listings to Pre Sale, For Sale, or Sold.
* Schedule a daily check for expired listings.
